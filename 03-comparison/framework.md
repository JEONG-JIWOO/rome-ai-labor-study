# 비교 프레임워크

## 비교의 원칙

1. **구조적 유추**: 직접 등치가 아닌, 구조적 패턴의 유사성/차이점 탐색
2. **다차원 교차**: 하나의 변수가 아닌 여러 차원을 교차 비교
3. **행위자 중심**: 거시 구조 + 각 행위자의 관점
4. **한계 명시**: 유추가 작동하지 않는 지점을 항상 기록
