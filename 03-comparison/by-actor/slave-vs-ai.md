# 노예 ↔ AI/로봇

> 가장 직관적이지만 가장 위험한 비교. 신중한 접근 필요.

## 유사한 기능적 역할

- 저비용 노동력으로서 기존 노동을 대체
- 소유자에게 이익 집중
- 생산성 향상에 기여

## 근본적 차이점

| 차원 | 노예 | AI |
|---|---|---|
| 존재론적 지위 | 인간 | 도구/프로그램 |
| 의식/의지 | 있음 | 없음 |
| 소비 | 소비자 | 비소비자 |
| 재생산 | 자연 재생산 가능 | 제조 |
| 저항 | 반란, 태업 가능 | 불가 |
| 해방 | 사회적 이동 가능 | 해당 없음 |
| 윤리적 문제 | 인간 착취 | 노동 대체 |
