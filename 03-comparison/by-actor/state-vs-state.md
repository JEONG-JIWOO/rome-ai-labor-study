# 로마 국가 ↔ 현대 국가

> 노동 대체에 대응하는 국가의 역량과 한계 비교.

## 로마 공화정/제정

- 원로원 과두정 → 제정으로 전환
- 제한된 관료 기구
- 곡물 배급, 토지법 등 직접적 개입
- 정보 수집·분석 역량 제한
→ [[political-crisis]], [[grain-dole]] 참조

## 현대 국가

- 민주주의 (다양한 형태)
- 발달된 관료 기구, 복지국가
- 정보·데이터 기반 정책 수립 가능
- 국제 협력 체계
→ [[ubi-debate]], [[political-polarization]] 참조
