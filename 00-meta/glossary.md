# 용어사전

## 로마/라틴어 용어

| 용어 | 원어 | 설명 |
|---|---|---|
| 라티푼디움 | latifundium | 대토지 소유 농장. 주로 노예 노동으로 운영 |
| 리베르티 | liberti | 해방노예. 법적 자유를 얻었지만 사회적 제약 존재 |
| 마누미시오 | manumissio | 노예 해방 절차 |
| 아논나 | annona | 곡물 배급 제도. 도시 빈민에게 무상/저가 곡물 공급 |
| 빌라 | villa | 농촌 저택 겸 농장. 노예 노동 기반 상업 농업의 중심 |
| 콜로누스 | colonus | 소작인. 후기 로마에서 점차 토지에 묶이는 존재가 됨 |
| 파트로누스 | patronus | 해방노예의 전 주인. 해방 후에도 의무 관계 유지 |
| 페쿨리움 | peculium | 노예가 관리할 수 있었던 재산. 해방 자금으로 활용 |
| 아게르 푸블리쿠스 | ager publicus | 공유지. 토지법 논쟁의 핵심 대상 |
| 프롤레타리 | proletarii | 재산이 없어 병역 면제된 최하층 시민 |

## 경제학 용어

| 용어 | 설명 |
|---|---|
| 포트폴리오 소득 | 여러 소득원을 조합하는 생존전략 (자작+임노동+수공업 등) |
| 긱 이코노미 | 단기·비정규 노동 기반 경제. 플랫폼 매개 |
| 총수요 | 경제 전체의 재화·서비스 수요 합계 |
| UBI | Universal Basic Income. 보편적 기본소득 |
| 노동시장 이중구조 | 안정된 1차 시장과 불안정한 2차 시장의 분리 |
| 자본 편향 기술 변화 | 기술 발전이 자본 소유자에게 더 많은 이익을 가져다주는 현상 |

## AI/자동화 용어

| 용어 | 설명 |
|---|---|
| AGI | Artificial General Intelligence. 범용 인공지능 |
| 자동화 대체율 | 특정 직업에서 AI/로봇으로 대체 가능한 업무 비율 |
| 보완적 자동화 | 인간 노동을 대체하지 않고 보완하는 자동화 |
| 대체적 자동화 | 인간 노동을 직접 대체하는 자동화 |
| 플랫폼 노동 | 디지털 플랫폼을 매개로 한 노동 (배달, 라이드헤일링 등) |
