# 방법론

## 비교 접근법: 구조적 유추 (Structural Analogy)

이 연구는 로마와 현대를 **직접적 인과 비교**가 아닌 **구조적 유추**로 비교한다.

### 핵심 원칙

1. **패턴 탐색**: 두 시대에서 노동 대체가 일으킨 구조적 패턴의 유사성/차이점을 탐색
2. **한계 명시**: 모든 비교에서 유추가 성립하지 않는 지점을 명시적으로 기록
3. **다차원 비교**: 단일 변수가 아닌 경제·정치·사회·제도적 차원을 교차 비교
4. **행위자 중심**: 거시 구조뿐 아니라 각 행위자(소농, 노예, 엘리트, 국가)의 관점에서 비교

### 유추의 한계

구조적 유추는 다음을 **하지 않는다**:
- 로마의 경험으로 현대를 예측하는 것
- "노예 = AI"와 같은 직접적 등치
- 하나의 역사적 사례에서 보편 법칙을 도출하는 것

유추가 유효한 범위와 무효한 범위는
[[false-analogies]]에서 지속적으로 관리한다.

### 자료 활용

- **로마 측**: 고고학적 증거, 문헌 사료, 생물고고학 데이터, 현대 학술 연구
- **현대 측**: 경제학 연구, 노동시장 통계, 정책 보고서, 기술 분석

### 비교 단위

| 차원 | 로마 측 | 현대 측 |
|---|---|---|
| 노동 대체 | 노예 노동력 | AI/자동화 |
| 피영향 주체 | 자유 소농 | 현대 노동자 |
| 자본 집중 | 라티푼디움/원로원 귀족 | 빅테크/초고자산층 |
| 국가 대응 | 곡물 배급, 토지법 | UBI, 재교육 정책 |
| 사회 이동 | 해방노예(liberti) | 재교육/전직 노동자 |
