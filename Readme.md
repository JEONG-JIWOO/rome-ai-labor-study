# Rome-AI Comparative Study

> 로마 노예 노동력 도입과 현대 AI/자동화의 구조적 비교 연구

## 핵심 질문

로마 공화정 후기~제정 초기에 대규모 노예 노동력이 도입되면서 자유 소농에게 일어난 변화와,
현대 사회에서 AI/자동화가 노동시장에 미치는 영향은 **구조적으로 어떤 점에서 비교 가능하고,
어떤 점에서 비교 불가능한가?**

## 방법론 요약

이 연구는 **구조적 유추(structural analogy)** 방법을 사용합니다.
직접적 인과 비교가 아니라, 두 시대의 노동 대체 현상이 공유하는 구조적 패턴을 탐색합니다.
동시에 유추의 한계를 명시적으로 관리합니다 (`03-comparison/false-analogies.md`).
